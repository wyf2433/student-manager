# 项目概述

## 项目定位

学生管理类 App(初中物理老师端)——面向初中物理老师的数字化班级管理工具。
个人使用,主打简单易用、快速记录。

## 技术栈

- 前端:微信原生小程序
- 网关:微信云函数(Node.js 薄转发层,免备案)
- 后端:Python FastAPI(香港服务器,纯公网 IP)
- 数据库:SQLite
- 协议:HTTP + API Key(无域名,无 HTTPS,云函数转发绕过白名单)

## 主要工作流

1. 后端开发:FastAPI + SQLite,REST API
2. 前端开发:微信小程序,2 Tab 架构(首页/学生)
3. 云函数:Node.js 转发层,携带 API Key
4. 联调与测试
5. 提交微信审核上线

## 功能模块(8 项)

1. 学生名册(手动添加 + 班级分组)
2. 考勤/请假/加扣分记录(底部快捷表单)
3. 工作留痕(拍照 + 分类,含实验课记录/作业反馈等)
4. 学生详情时间线(记录聚合)
5. Excel 成绩导入(后端解析 + 预览确认,优先识别物理列)
6. 一句话速记 + 今日概览
7. 课表 + 今日课程 + 临时调课
8. 作业布置 + 状态流转 + 留痕关联
